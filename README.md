## Usage

### Installation

```sh
    npm install
```

### Development

Launch development server.

```sh
    npm start
```

### Build

Compile TypeScript app and copy index.html to the `dist` folder.

```sh
    npm run build
```

### Deploy

Deploy `dist` folder to GitHub Pages.

```sh
    npm run deploy
```

You can see the result in <https://jefersonla.github.io/rifa-typescript-app/>.
